# 📋 Sobre o Projeto

O **Prompts Concursando IA** reúne uma coleção de prompts especializados para auxiliar candidatos a concursos públicos brasileiros em diferentes etapas da preparação. Todos os prompts iniciam solicitando informações essenciais, como **nome do concurso, banca organizadora, nível de escolaridade, cargo pretendido e confirmação do edital vigente**, garantindo respostas mais contextualizadas. Além disso, foram desenvolvidos com base em estratégias de aprendizagem apoiadas por evidências, como **Active Recall**, **Repetição Espaçada**, **Prática Deliberada** e outras técnicas que favorecem retenção e desempenho.

# 🚀 Como Usar

1. 📂 Escolha o cenário desejado na tabela acima.
2. 📄 Abra o arquivo `.md` correspondente.
3. 📋 Copie o prompt completo.
4. 🤖 Cole em uma IA generativa (ChatGPT, Claude, Gemini, Copilot, DeepSeek, entre outras).
5. ✏️ Informe os dados solicitados pelo prompt, incluindo o **concurso**, **banca**, **cargo**, **nível** e confirme que o **edital é o mais recente**.

> **⚠️ Importante:** Nenhum prompt substitui o estudo oficial por materiais atualizados e a leitura da legislação.

# 🤝 Contribuições

Contribuições são muito bem-vindas!

Caso deseje adicionar um novo prompt:

- Faça um **Fork** do projeto;
- Crie uma nova branch;
- Adicione seu arquivo `.md`;
- Envie um **Pull Request** com uma descrição clara das alterações.

Também é possível abrir uma **Issue** para sugerir melhorias, relatar problemas ou propor novos cenários.

### Padrão obrigatório para novos prompts

Todo prompt deve iniciar solicitando obrigatoriamente:

- 📌 Nome do concurso
- 🏛️ Banca organizadora
- 🎓 Nível (Fundamental, Médio ou Superior)
- 💼 Cargo
- 📑 Confirmação de que o edital utilizado é o mais recente
